Find the Opportunity URL

Before opening the campaign manager, you need the direct URL to the opportunity — the exact page a student should land on after clicking the link.

What is the destination URL? It's the public link to the opportunity — a company's careers page, an application form, or a Knock listing. Copy it from your browser's address bar.
🔒 absa.co.za/careers/grad-programme-2025
ABSA Grad Programme 2025
Applications open for the 2025 Graduate Programme across Finance, Technology and Operations.
Apply Now
Before you continue
I have the opportunity URL copied from my browser
e.g. https://absa.co.za/careers/grad-programme-2025
The URL goes directly to the opportunity — not a homepage or login page
The page is publicly accessible — students can open it without logging in

Field Format Example Notes
Organisation orgname absa standard_bank Lowercase, underscores for spaces
Group Name university_faculty wits_commerce uct_engineering University: wits, uct, stellenbosch, uj, up
Channel dropdown whatsapp webapp personal Select from the dropdown — never type manually
Wave wave#_type wave1_core wave2_follow core = first post · follow = reminder · final = last call
Before you continue
Organisation is lowercase with underscores — no capitals or special characters
Group name follows the format: university_faculty
e.g. wits_commerce, uct_engineering, stellenbosch_law
Channel is selected from the dropdown — not typed in
Wave follows wave#_type format and has not been reused for a different opportunity
